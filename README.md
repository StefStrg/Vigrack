# Vigrack
Command Line Tool for cracking vigenere ciphers using frequency analysis.
##Installation 
To use vigrack as a command, clone the repo and run the install script:

chmod +x install.sh

./install.sh
NOTE: This is a base version to use as a test, this will not work every time, because we use frequency analysis and we do not provide a key for the decryption. Also, we need a big sample to have a more successful outcome, for example if you have an issue with finding the key or the plaintext message keep in mind that a small sample will not be accurate. Make sure to not have any whitespaces.
